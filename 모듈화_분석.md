### 1-1. `measin_constants.py` (61줄) — 공통 상수 + CSS 셀렉터 분리

**배경:** `eco_input.py`와 `eco_check.py`에 동일한 상수(SKIP_VOL_AND_SPEED, SKIP_SPEED_ONLY, sm3_items 등)가 중복 정의되어 한쪽만 수정 시 불일치 위험이 있었음.

**적용 내용:**
```python
# measin_constants.py — 공통 상수
SKIP_VOL_AND_SPEED = { "매연", "황산화물", ... }
SKIP_SPEED_ONLY    = { "먼지", "비소화합물", ... }
HEAVY_METALS       = { "비소화합물", "크로뮴화합물", ... }
SM3_ITEMS          = { "먼지", "구리화합물", ... }
DUST_SKIP_FIELDS   = { "표준산소농도", "실측산소농도", ... }

# 2차 추가: CSS 셀렉터 (탭2 필드)
SEL_DATE = "#meas_end_dt"
SEL_START_TIME = "#meas_start_time"
SEL_END_TIME = "#meas_end_time"
SEL_O2_STD = "#basis_o2c"      SEL_O2_MEAS = "#meas_o2c"
SEL_GAS_VOL_PRE / POST / SEL_MOISTURE / SEL_GAS_TEMP / SEL_GAS_SPEED
```

**사용처:**
- `eco_input.py` → `from measin_constants import SKIP_VOL_AND_SPEED, SKIP_SPEED_ONLY, HEAVY_METALS, SM3_ITEMS as sm3_items`
- `eco_check.py` → `from measin_constants import SKIP_VOL_AND_SPEED, SKIP_SPEED_ONLY, DUST_SKIP_FIELDS, SM3_ITEMS as sm3_items`

### 2차: 중복 제거 + 셀렉터 상수화 (현재 세션)

### 1-5. `wait()` → `selenium_utils.py`로 통합

**배경:** eco_input.py와 eco_check.py 양쪽에 `def wait(sec): time.sleep(sec)` 동일 함수 존재.

**적용:** `selenium_utils.py`에 `wait()` 추가, 양쪽에서 import. 로컬 정의 삭제.

### 1-6. `clean()` / `clean_leading_mark()` → `data_utils.py`로 통합

**배경:** eco_input.py의 `clean(s)`와 eco_check.py의 `clean_leading_mark(s)`가 동일 로직(선행 ×, ✕, 쉼표 제거)이었으나 이름만 달랐음.

**적용:** `data_utils.py`에 `clean_leading_mark()` 추가.
- `eco_input.py` → `from data_utils import clean_leading_mark as clean`
- `eco_check.py` → `from data_utils import clean_leading_mark`

### 1-7. CSS 셀렉터 10개 → `measin_constants.py`에 상수화

**배경:** `#meas_end_dt`, `#meas_start_time`, `#basis_o2c` 등 탭2 필드 셀렉터 10개가 eco_input.py와 eco_check.py 양쪽에 하드코딩되어, 사이트 HTML 변경 시 양쪽 모두 수정해야 했음.

**적용:**
```python
# measin_constants.py에 추가
SEL_DATE         = "#meas_end_dt"
SEL_START_TIME   = "#meas_start_time"
SEL_END_TIME     = "#meas_end_time"
SEL_O2_STD       = "#basis_o2c"
SEL_O2_MEAS      = "#meas_o2c"
SEL_GAS_VOL_PRE  = "#meas_gas_fvol"
SEL_GAS_VOL_POST = "#meas_gas_fvol_o2_aft"
SEL_MOISTURE     = "#meas_humd_vol"
SEL_GAS_TEMP     = "#gas_meter_temper"
SEL_GAS_SPEED    = "#meas_fspd"
```
